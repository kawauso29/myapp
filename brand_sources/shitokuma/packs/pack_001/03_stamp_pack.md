# 03_stamp_pack.md

## このパックのコンセプト
雨で予定がずれた日の連絡を、しとくまの半目トーンで丸く伝える実務パック。  
謝罪だけで終わらず、代替案やフォローの一言まで返せる構成にする。  
在宅ワークの「遅れ・調整・お礼」を8枚で一連の流れとして使えるようにする。

## 表現レイヤー
Core Layer + Work Layer（在宅ワーク連絡特化）

## 採用しない要素（派生パックへ）
- 夢演出（雲ベッド、しずく星、霧強め）は Dream Layer 派生で採用
- 季節イベント（梅雨祭り、台風実況など）は Seasonal Layer 派生で採用

## 個別書き出しルール
- 1画像1スタンプ
- 文字は中央上部に配置
- 半目〜伏し目・小さい口を全カットで維持
- 雨粒・傘モチーフは控えめに添える
- パック一覧画像を必ず参照して揺れを防ぐ
