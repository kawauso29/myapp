# 03_stamp_pack.md

## このパックのコンセプト
雨の日の「遅れる・確認中・助かった」を、しとしとねこの半目トーンでやわらかく伝える実用パック。
気まずい連絡を必要以上に重くせず、相手の不安を下げる言葉だけで構成する。
仕事連絡にも日常チャットにも使える、低刺激で丁寧な8枚に統一する。

## 表現レイヤー
Core Layer + Work Layer を採用する。
重めの半目・小さな口・水色しずくタグを固定し、PC/スマホ/傘など日常小物で利用シーンを明確化する。

## 採用しない要素(派生パックへ)
- Dream Layerの浮遊演出や幻想系ライティング
- テンションを上げるジャンプや大笑い表情
- 強い煽り文句や断定的な命令口調

## 個別書き出しルール
- 1画像1スタンプ
- 文字は中央上部に配置
- 小さく表示されても読める漢字を維持する
- 雨粒・雨雲モチーフは主役を邪魔しないサイズにする
- パック一覧画像を参照し、表情・線・文字スタイルの揺れを防ぐ
