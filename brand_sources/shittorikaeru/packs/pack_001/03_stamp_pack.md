# 03_stamp_pack.md

## このパックのコンセプト
梅雨どきの仕事連絡で生まれる「返信遅れ・待たせる気まずさ」を、しとカエルの半目と小さな気づかいでやわらげる8枚パック。
雨粒や透明傘などの湿度モチーフを控えめに添え、責めずに伝える連絡トーンへ寄せる。
忙しい相手に配慮しながら、短文で状況共有できる設計にする。

## 表現レイヤー
Core + Work Layer を採用。Dream Layer は採用しない。

## 採用しない要素(派生パックへ)
- 虹・あじさいの華やかな演出
- 寝落ちや夢見心地の背景表現
- 季節イベント(七夕・夏祭り)の装飾

## 個別書き出しルール
- 1画像1スタンプ
- 文字は中央上部に配置
- パックシート画像を必ず参照して揺れを防ぐ
