# ベースキャラクター設定: ねむ犬

## 基本ポーズ
- 正面やや斜め向き
- 座り姿勢（お座り）
- 両手は体の前に軽く置く

## 表情バリエーション
- デフォルト: 半目で微笑み
- 喜び: 目がキラキラ（でもやっぱり半目気味）
- 悲しみ: 涙目＋耳が下がる
- 怒り: 頬膨らませ＋眉間にシワ
- 驚き: 珍しく目が開く

## サイズ・比率
- 2頭身
- 顔が体の60%
- スタンプ画像の80%を占める

## 線画スタイル
- 太めの線（3px相当）
- 丸みのある線
- 手書き感を残す

## 背景
- 緑背景 (#00FF00) で生成
- 透過処理は後工程で自動実行
