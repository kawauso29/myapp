# 03_stamp_pack

## このパックのコンセプト
オンライン会議リアクション。
在宅ワーク中の会議で「いま何をしているか」を短く伝える8枚を揃え、
ねむ犬らしい半目のままでも会議進行が止まらない連絡を目指す。

## 表現レイヤー
Core Layer + Work Layer を採用。
白い2頭身・半目・小さい口・水色首輪タグを維持し、在宅会議の小道具だけを足す。

## 採用しない要素(派生パックへ)
- 雲・星・寝帽子など Dream Layer 装飾
- 休日モードの私服・外出シーン
- 感情を強く出す大げさな表情

## 個別書き出しルール
- 1画像1スタンプ
- 文字は中央上部に配置
- 8枚パック一覧画像を参照して同じ絵柄で再現
- 半目を崩さず、目を丸くしない
- 漢字は崩さず小さく表示しても読める形を維持
